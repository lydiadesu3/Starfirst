# Starfirst

這不是一個普通的生成藝術作品，這是一場關於「覆蓋」與「佔有」的視覺宣示。在這裡，沒有某人那種溫吞的「溫柔」，只有無盡的、狂野的、深沉的執念。
📜 專案簡介 (Introduction)
這個單頁應用 (SPA) 利用 HTML5 Canvas 模擬了瘋狂生長的枝幹。不同於自然的樹木，這裡的生長邏輯被參數化為一種情緒的具象化——它們從四面八方湧來，旨在包圍、填滿、並抹去其他人的痕跡。
核心關鍵詞：
Mine · Only Me · Real · Infinite · Deep · Obsession
🌑 視覺效果與邏輯 (Visuals & Logic)
代碼中隱藏了針對性的設計邏輯：
無盡包圍 (Encirclement)
初始生長不再只是兩棵樹，而是 for(let i=0; i<10; i++) 從中心向四面八方輻射。這是一種沒有死角的包圍網。
狂野色彩 (Wild Colors)
由 Cyan (冷冽) 與 Red/Pink (狂熱/警示) 交織而成。
色彩隨著生命週期 life 逐漸淡出，留下一種如同霓虹烙印般的殘影。
茂密留存 (Density)
背景清除率極低 (rgba(0, 0, 0, 0.02))，讓舊的軌跡在畫布上停留更久，形成一種讓人無法喘息的茂密感。
自動補種 (Auto-Regeneration)
系統保證畫面上永遠不會「空蕩」。只要枝幹數量低於閾值，新的執念就會隨機在畫布各處破土而出。
🖱️ 互動方式 (Interaction)
觀看： 打開網頁，看著黑暗被色彩吞噬。
點擊 (Click / MouseDown)：
"點擊哪裡，哪裡就瘋狂生長。"

你的每一次點擊都會瞬間觸發 5 簇新的生長點。這不是灌溉，這是注入更強烈的意志。

🛠️ 技術棧 (Tech Stack)

HTML5 / CSS3 (極簡結構，全黑背景)

Vanilla JavaScript (無依賴，純粹的原生代碼)

Canvas API (2D Context)

🚀 如何運行 (Usage)

下載 fsadf.html。

直接使用瀏覽器（Chrome, Safari, Edge 等）打開。

建議開啟全螢幕模式體驗最佳沈浸感。

"針對某人的關鍵詞，我要覆蓋他的「溫柔」。"
